Data Analysis and Visualization using R (1)
Course code: BFVH19DAVUR1

This course teaches the fundamentals of data analysis, data exploration and data visualization using the programming language R

Presentation 1: Toolbox

This is the general introduction of the course. It gives a brief overview of the tools in the R toolbox: R, RStudio, RMarkdown and Swirl. For the pdf version, click here.

Presentation 2: Basic R

The basic datatypes and functionality of R are explored, as well as some plotting techniques. For the pdf version, click here.

Presentation 3: Complex Datatypes

You will get to know the more complex of the R data types: Factor, List, Dataframe. For the pdf version, click here.

Presentation 4: Functions

A variety of function-related aspects of R are presented: Most important R functions and creating functions. For the pdf version, click here.

Presentation 5: Dataframe manipulations

Here, all functions and techniques used fo manipulating dataframes (and lists) are discussed. These include 'with', 'subset', 'apply', 'aggregate', 'split'. For the pdf version, click here.

Example use cases

Some example use cases with dataframes: selections, apply-ing and processing embedded dataframes.
