Application Design

This course deals with more advanced Java topics such as JUnit testing, multithreading and lambdas and the Streams API, as well as OO oriented design principles and design patterns.

Minor project Application Design

In this project, the Application Design course is put in practice. Students work on real-world problems in bioinformatics. Central is the development of a Spring-based web application. Agile, Scrum, ticketing, use cases, user stories and logging are also introduced.
