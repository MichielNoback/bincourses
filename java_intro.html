Introduction to Java programming Course code: BFVH3JAVA

During this introductory course you will learn to program in Java and create a simple Object-Oriented design. We use IntelliJ Idea IDE as development platform.

The setup of the course has changed somewhat. You will need to study a small amount of theory and carry out some activity before each lesson. During class, the more challenging parts of theory will be discussed and quizzed, and we will work on programming challenges. The amount of teacher presentation time will be reduced to a bare minimum.
You should study the course program carefully in advance! All work will be carried out using the repo JavaIntroProgrammingAssignments as starting point. You should clone this repo as soon as possible.
The course is graded only through the final test at the end, a practical test where the course gitbook will be available offline.

Gitbook

The presentations have been replaced by a gitbook. This gitbook will also be available at the test. Take a look at the course program to see what is the topic of each lesson.

Training exam

Clone the repo, and work on the assignments in branch WithoutSolutions. When you are done, run the test class to find out how you have done. If you are really stuck, you can check out branch 'master' to see the solutions. (In IntelliJ via, VCS -> git -> Branches).
