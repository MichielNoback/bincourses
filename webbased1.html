Webbased information systems 1 Course code: BFVH4WBI1

This course will introduce you to several aspects related to designing and creating a dynamic web application using Java web technology, Thymeleaf and MySQL for the back-end and a (tiny) bit of HTML/CSS for the front-end.

Download the DbUtils jar here (used in section on DB access). Add it to build.gradle using this: compile files('/path/to/lib/DButils-1.0.2.jar')
