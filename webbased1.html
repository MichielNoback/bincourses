Webbased information systems 1 Course code: BFVH4WBI1

This course will introduce you to several aspects related to designing and creating a dynamic web application using Java web technology and MySQL for the back-end and Javascript/HTML/CSS for the front-end.

Presentation 1

General introduction on the essentials of web applications: request, respons, http, ports etc.

Presentation 2

Getting started with web development in InteliJ Idea and introduction of MVC.

Presentation 3

Implementing the Model View Control paradigm in a Java web application backend.

Presentation 4

User authentications, HTTP sessions and how to work with view templating using Java Server Pages (JSP).

Presentation 5

Interacting with a MySQL database backend as data layer. And a little bit design concepts (Singleton, Program against Interfaces)

Presentation 6

A bit of front-end technology: Javascript and jQuery.

Training exam

Download the training exam as zip archive of a Gradle managed project. Take 2 hours to solve the problems, using only the provided cheat sheet.
